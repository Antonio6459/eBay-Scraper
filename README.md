# eBay-Scraper
Scans 8 international eBay domains simultaneously to find the absolute cheapest product listings in seconds.

Find the Cheapest Deals Across 8 Global eBay Markets Instantly!

Stop manually switching sites, translating text, and converting currencies. This lightweight, encrypted desktop application automatically finds the absolute lowest prices across international eBay domains in seconds.

What You Get:

Global Search: Scans eBay US, UK, IT, FR, DE, ES, AU, and CA all at once.

Smart Parsing: Automatically translates search terms and handles European currency conversions.

Top 5 List: Cleans the data and displays the 5 cheapest live listings with direct links.

Instant Start: No python setup or browser installations required.

How to Use It:

1 -> Download and unzip the folder.

2 -> Double-click Scrappy.exe.

3 -> Type your product and see the deals roll in
